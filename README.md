#Remove Background Desktop

Dekstop remove backgorund menggunakan python, sebelum menggunakan perlu melakukan instalasi python dahulu 

Jika memakai CPU maka:
pip install "rembg[cpu]"
pip install pillow
pip install tkinterdnd2

Jika memakai GPU versi (NVIDIA + CUDA) maka:
pip install "rembg[gpu]"
pip install pillow
pip install tkinterdnd2
